# EBSOC Addon
#### ebsoc.co.kr의 일부 부족한 기능을 보완하기 위해 제작되었습니다.
#### 주의: 처음으로 만들어보는 Chrome Extension입니다. PR는 언제나 환영합니다.

## EBSPlayer
#### player.ebsoc.co.kr의 내용을 일부 추가합니다.
- 동영상 제목을 표시합니다.
- [ 새 탭에서 영상 열기 ] 를 통해 새 창에 동영상을 표시합니다. (진도율은 추가되지 않으나, 놓친 부분을 보기 위해서는 필요하다고 생각하였습니다.)
